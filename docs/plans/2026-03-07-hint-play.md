# Hint Play Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 为当前出牌玩家增加“提示”按钮，点击后自动选中一组合法可出的牌，并在同一局面下连续点击时在全部可出方案之间轮换。

**Architecture:** 在 `shared` 新增一个纯函数统一枚举并排序所有可出方案，客户端只负责维护提示轮换索引和把某一组 `CardPlay.cards` 写入选牌状态。这样出牌规则仍集中在 shared，前端不复制一套规则逻辑。

**Tech Stack:** TypeScript, React 19, Zustand, Vitest, pnpm workspace
